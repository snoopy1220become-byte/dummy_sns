# GAS (Google Apps Script) Web App - 架空SNS「Echoes」

Google Apps Script (GAS) の `doGet` で動作し、現代的で美しい UI/UX を持つ架空のSNSアプリ「Echoes」です。

## 主な機能と特徴
1. **モバイルフレンドリー & レスポンシブ**
   - デスクトップ環境ではスマートフォン型のデバイスモックアップフレーム内に美しく配置され、スマートフォンからは画面いっぱいに表示されます。
2. **引っ張ってリロード (Pull to Refresh)**
   - タイムラインの最上部でさらに下にスワイプ（またはマウスでドラッグ）すると、カスタムのローディングインジケーターが出現し、1.5秒間のローディング後に新規投稿をモックで追加して自動スナップバックします。
3. **高機能インタラクション**
   - **投稿**、**いいね！（ハートアニメーション）**、**コメント返信**がリアルタイムに反映されます。
   - 3名のサンプルユーザーのアカウント切り替えが可能で、複数人の会話を1人でシミュレートできます。
4. **GASへのデプロイ & ローカルプレビュー両対応**
   - ローカルPCで `Index.html` をダブルクリックするだけで、`localStorage` を用いたモックデータベースによりオフラインで動作確認が可能です。
   - GASにコピペすれば、GAS環境のAPI経由で動くWebアプリケーションになります。

## ファイル構成
- `Code.js` : GASのバックエンドロジック (doGet, getPosts, createPost, likePost, addComment)
- `Index.html` : UI、CSS、およびフロントエンド制御JS

## GASへの導入手順
1. Google ドライブで「新規」>「その他」>「Google Apps Script」を開きます。
2. デフォルトで作成される `コード.gs` の内容を、このフォルダの `Code.js` の内容に書き換えます。
3. エディタ左側の「＋（ファイルを追加）」ボタンから「HTML」を選択し、ファイル名を `Index` にします（`Index.html` になります）。
4. 新しく作成された `Index.html` の中身を、このフォルダの `Index.html` の内容にすべて書き換えます。
5. エディタ右上にある「デプロイ」>「新しいデプロイ」をクリックします。
6. 種類の選択で「ウェブアプリ」を選び、以下を設定します：
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員**（または「自分のみ」）
7. デプロイを実行し、発行された「ウェブアプリのURL」にアクセスします。
